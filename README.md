# Magic Framework

> 轻量级PHP框架

## 框架特色

* 基于Composer，安装、替换组件使用更方便
* 去除模板编译层，前端使用Vuejs代替渲染，让页面解析更快
* 更多等你发现 ...

## 开始使用

1、下载代码

	git clone https://github.com/menghuanplus/magic.git

2、安装依赖

	cd m && composer update

3、运行

	cd public && php -S 127.0.0.1:3000

[访问 http://127.0.0.1:3000](http://127.0.0.1:3000)

Enjoy it ...
